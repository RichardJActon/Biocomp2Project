Database Importer

home

Synopsis

    Takes the data output by the genbank file parser into 3 "|" separated files and imports each into their corresponding tables in the database. This script also builds the index for the database following the import of the data.

Arguments

  • File named "loci.txt" containing columns: Genbank_Accession, Locus_GI, DNA_seq, Product_Name, CDS_translated, Reading_frame separated by "|" with each row on a new line "\n".

  • File named "chromloc.txt" containing columns: Genbank_Accession, Location_Name separated by "|" with each row on a new line "\n".

  • File named "exons.txt" containing columns: Genbank_Accession, StartPosition, EndPosition, Exon_ID separated by "|" with each row on a new line "\n".

Returns

  • The data is loaded into the database.
