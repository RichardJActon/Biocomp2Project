Developers:

This website was designed and developed by Gabriele Nocchi, Dolapo Ajayi and Richard Acton.


What this website offers:

This website provides the user with easy access to detailed genetic information about human chromosome 17. Explore our database using one of four widely used gene id(s). Extract the amino acid sequence for a given gene. Find out about codon usage frequencies within specific genes and more...



Development Strategy:

The construction of this website was divided in 3 parts, nameley the front end, the middle section and the database.

The Database:

The database was headed up by Richard Acton. Richard was in charge of extracting the deluge of information on the human chromosome 17 which was stored in a text file. He designed the database into which all the data extracted was stored and did so in a way that made it possible to query and search the database efficiently.

The Middle section:

The middle section was overseen by Gabriele Nocchi. Gabriele managed the functions that provide the information to the user such as codon frequency usage info and the highlighting of coding regions in the DNA sequence. The resulting form of the information converted from data by Gabriele's calculations made displaying this information all the more straightforward.

The frontend:

The front end was rendered by Dolapo Ajayi. Dolapo handled the unpacking of the information extracted from the database and provided by the middle layer. He worked on the aesthetic aspects of the website and the mobility through the website to provide a user-friendly experience.
