What this website offers

This website provides the user with easy access to detailed genetic information about human chromosome 17. Explore our database using one of four widely used gene id(s). Extract the amino acid sequence for a given gene. Find out about codon usage frequencies within specific genes and more...



Searching through chromosome 17

From the home page you are provided with the option of searching for a gene using one of four identifiers, namely:

  1. Gene ID e.g. AB007119
  2. Protein product name e.g. keratin 12
  3. Genbank accession e.g. 60391373
  4. Chromosomal location e.g. 17q12

If you know beforehand the Gene ID or Genbank accession number for the gene you wish to search for as in the cases of options 1 and 3, you will obtain a single result of the specific gene.

Alternatively you may be wanting to search for a gene using the name of the protein the gene translates to, in which case you simply type in the term associated with the gene/protein e.g. hydrolase.

There is also the option of simply sifting through the entire chromosome by leaving the search box blank and pressing search.
